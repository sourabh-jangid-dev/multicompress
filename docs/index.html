Get started in 3 steps

No Python needed — FFmpeg ships inside.

  1. Download the latest MultiCompress.zip from the Releases page.
  2. Unzip it and run MultiCompress.exe — no installation required.
  3. Drag your files in, pick a preset, hit Compress All. That's it. 🎉

Built with Python · FFmpeg · Pillow · PyMuPDF — free and open source under the MIT license.

Made for creators who shouldn't have to pay to make a file smaller.
